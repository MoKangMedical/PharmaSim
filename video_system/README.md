## 📋 系统概述

这是一个完整的视频和语音制作系统，为PharmaSim项目提供专业的视频制作能力。

## 🚀 快速开始

```bash
# 进入系统目录
cd ~/Desktop/PharmaSim/video_system

# 查看帮助
./run.sh help

# 生成测试语音
./run.sh voice --text "PharmaSim AI-Powered Drug Launch Simulation"

# 启动Web界面
./run.sh web

# 运行测试
python3 test_system.py
```
