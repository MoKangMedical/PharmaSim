GET /api/drugs.json All drug profiles with scores & metadata

Description

Returns the complete list of 14 approved drugs in the simulation database. Each entry includes drug name, generic name, Chinese name, company, therapeutic area, indication, approval status, NMPA ID, and associated simulation scores.

Response Fields

FieldTypeDescription
idnumberUnique drug identifier
namestringBrand name
generic_namestringINN generic name
chinese_namestringChinese drug name
therapeutic_areastringCategory (e.g., oncology, metabolism)
indicationstringPrimary approved indication
companystringManufacturer / sponsor

Example Response

[ { "id": 1, "name": "Keytruda", "generic_name": "pembrolizumab", "chinese_name": "帕博利珠单抗", "therapeutic_area": "肿瘤/免疫", "indication": "非小细胞肺癌" }, ... ]
GET /api/scores.json Dimension-level scores for every drug

Description

Returns detailed scoring data across all 7 dimensions (and 28 sub-dimensions) for each simulated drug. Includes composite scores, dimension weights, and agent consensus metrics.

Response Structure

[ { "drug_id": 1, "composite_score": 0.691, "dimensions": { "epidemiology": 0.72, "clinical": 0.81, "market": 0.65, "pricing": 0.58, "pharmacology": 0.73, "pharmacoeconomics": 0.61, "insurance": 0.67 } }, ... ]
GET /api/summary.json High-level simulation summary & rankings

Description

Provides an aggregated summary of all simulation runs. Includes total drug count, average scores, top-performing drugs by composite score, recommendation distribution (approve / conditional / reject), and therapeutic area breakdowns.

Key Fields

{ "total_drugs": 14, "avg_composite_score": 0.633, "recommendations": { "approve": 6, "conditional": 5, "reject": 3 }, "top_score": 0.704, "simulation_count": 2847 }
GET /api/agents.json AI agent network definitions & attributes

Description

Returns metadata about the 1,800 AI agent network: 400 doctor agents, 1,000 patient agents, and 400 expert agents. Each agent has attributes including specialty, region, influence score, and network connectivity.

Response Structure

{ "total_agents": 1800, "types": { "doctors": 400, "patients": 1000, "experts": 400 }, "network_model": "Watts-Strogatz", "avg_connections": 8.3 }
GET /api/connections.json Agent network edge list & graph topology

Description

Full edge list for the Watts-Strogatz small-world network connecting all 1,800 agents. Used for information diffusion modeling and visualization in the Pipeline view.

Response Structure

[ { "source": 0, "target": 1, "weight": 0.82 }, { "source": 0, "target": 5, "weight": 0.64 }, ... ]

GET /api/epidemiology.json Disease burden & epidemiological data

Description

Chinese epidemiological data by disease area: prevalence, incidence, DALY, mortality rates, patient population size, and demographic distributions. Used as input for the epidemiology scoring dimension.

Response Structure

[ { "disease_area": "非小细胞肺癌", "prevalence_cn": 820000, "incidence_per_100k": 57.3, "mortality_rate": 0.46, "dalys_millions": 12.4 }, ... ]
GET /api/pipeline.json Drug pipeline stages & trial timelines

Description

Pipeline data for 8 drugs currently in simulation. Includes current phase, trial start/end dates, NDA filing status, estimated approval timeline, and competitive landscape analysis.

Response Structure

[ { "drug_id": 1, "name": "Tirzepatide", "phase": "Phase 3", "indication": "T2D / Obesity", "est_approval": "2025-Q4", "nda_filed": true }, ... ]
POST /api/simulate Run custom simulation (coming soon)

Description

Submit custom drug parameters to run a simulation against the 1,800-agent network. Returns predicted composite scores, dimension breakdowns, and recommendation consensus. Currently in beta — available through the Dashboard UI.

Request Body

{ "drug_name": "Custom Drug A", "therapeutic_area": "Oncology", "phase": "Phase 2", "params": { "efficacy_score": 0.8, "safety_profile": 0.75, "price_range": "high" } }

Free All simulation endpoints are free to use with no rate limits.
