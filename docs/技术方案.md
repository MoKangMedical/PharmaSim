## 一、系统概述

### 1.1 项目背景

药品上市是医药企业最核心、风险最高的决策之一。传统方式依赖：
- 医生调研（耗时3-6个月，样本有限）
- 历史类比分析（缺乏可比性）
- 专家经验判断（主观偏差大）

**核心痛点：**
- 一个新药从临床到上市，市场预测偏差可达 30%-50%
- 竞品突然上市、医保政策变化等事件难以预判
- 不同定价策略、推广力度的效果无法提前验证

### 1.2 解决方案

PharmaSim 通过构建**多智能体仿真系统**，模拟药品上市后各利益相关方的行为反应：
- **医生 Agent**：模拟处方决策行为
- **患者 Agent**：模拟就医、用药依从性行为
- **医保 Agent**：模拟医保谈判和报销政策影响
- **渠道 Agent**：模拟药房、电商平台的进货和推荐行为
- **竞品 Agent**：模拟竞品企业的市场应对策略

### 1.3 与 Aaru 的差异定位

| 维度 | Aaru | PharmaSim |
|------|------|-----------|
| 行业 | 通用（商业/政治/政府） | 深耕医药医疗 |
| 数据源 | 交易数据、选票数据 | 处方数据、医保数据、临床数据 |
| Agent 类型 | 通用消费者 | 医生/患者/医保/渠道 |
| 输出 | 通用市场洞察 | 药品上市KPI预测（处方量、渗透率、收入） |
| 合规要求 | 美国隐私法 | 中国《药品管理法》《个人信息保护法》 |

#### 模块2：行为决策模型

每个 Agent 的决策基于以下框架：

```
决策输入：
├── 药品属性（疗效/安全性/价格/剂型/品牌）
├── Agent 个人特征（经验/偏好/约束）
├── 社交影响（同行推荐/KOL意见/患者反馈）
├── 外部环境（医保政策/竞品动态/舆论）
└── 历史行为模式（从真实数据中学习）

决策过程：
├── 信息收集（模拟搜索/咨询行为）
├── 选项评估（多因素加权）
├── 社交协商（与相关Agent交互）
└── 最终决策 + 不确定性表达

决策输出：
├── 处方选择（医生）
├── 用药决策（患者）
├── 报销政策（医保）
└── 渠道策略（药房/电商）
```

#### 模块3：时间演化引擎

模拟药品上市后 1-24 个月的动态演化：

```
Month 0:  上市发布
  ↓
Month 1:  早期采纳者（学术型医生先试用）
  ↓
Month 3:  口碑形成（患者社区讨论/医生同行交流）
  ↓
Month 6:  医保谈判窗口（可能纳入/不纳入）
  ↓
Month 12: 市场稳定期（渗透率趋稳/竞品反应）
  ↓
Month 24: 长期格局（市场分额稳定/续约谈判）
```

| 层级 | 技术选型 | 理由 |

| 数据可视化 | Plotly + ECharts | 交互式图表 |

## 五、关键技术挑战与解决方案

### 挑战1：Agent 行为一致性

**问题：** LLM 输出具有随机性，同一个 Agent 在不同时间可能做出矛盾决策。

**解决方案：**
- 为每个 Agent 维护**记忆系统**（向量数据库存储历史决策）
- 决策时注入 Agent 的历史行为作为约束
- 设置行为参数的确定性边界（温度降低 + 规则约束）

### 挑战2：规模化性能

**问题：** 模拟 10,000+ Agent 的 LLM 调用成本和时间。

**解决方案：**
- Agent 分层：核心 Agent（高精度LLM）+ 普通 Agent（规则+小模型）
- 批量推理优化
- 增量模拟：只在关键时间点触发全量模拟
- 缓存相似 Agent 的决策结果

### 挑战3：数据获取

**问题：** 处方数据、医保数据敏感且难以获取。

**解决方案：**
- Phase 1：使用公开数据（PDB、药智网、CDE）
- Phase 2：与药企合作，用脱敏数据训练
- Phase 3：建立数据合作网络（医院、医保局、第三方数据商）

### 挑战4：结果可信度

**问题：** 客户为什么相信 AI 模拟的结果？

**解决方案：**
- 回测验证：用已上市药品的真实数据背书
- 置信区间：每个预测都给出不确定性范围
- 可解释性：每个决策都能追溯到具体 Agent 的推理过程
- 渐进式信任：先做辅助参考，逐步替代传统调研

## 六、与现有 DrugMind 项目的协同

你的桌面已有 DrugMind 项目（药物研发方向），PharmaSim 可以与其形成互补：

```
DrugMind（上游）              PharmaSim（下游）
┌──────────────┐            ┌──────────────┐
│ 靶点发现     │            │ 上市预测     │
│ 分子设计     │ ────→     │ 定价策略     │
│ 临床试验设计 │            │ 市场渗透     │
│ 药物重定位   │            │ 竞品应对     │
└──────────────┘            └──────────────┘
     研发阶段                    商业化阶段
```

**协同点：**
1. DrugMind 的临床数据可以直接喂给 PharmaSim 做上市预测
2. PharmaSim 的市场反馈可以指导 DrugMind 的研发优先级
3. 共享数据基础设施和 LLM API
